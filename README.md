# Auto-PR Agent 🚀

An AI-driven multi-agent system built to automatically review legacy code, refactor technical debt, and run closed-loop verification tests before opening a Pull Request. Powered by **Xiaomi MiMo-V2.5** and OpenClaw.

## Features
- **Deep Reasoning Review**: Analyzes code structures and maps complex dependencies.
- **Automated Refactoring**: Rewrites inefficient code blocks automatically based on modern standards.
- **Closed-Loop Testing**: Runs local test suites to ensure zero breakage before submission.

## Tech Stack
- **LLM/API**: [Xiaomi MiMo API](https://platform.xiaomimimo.com/docs/en-US/welcome) (MiMo-V2.5-Pro)
- **Framework**: OpenClaw / Python-based custom agent orchestration
